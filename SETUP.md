## Настройка для продакшена

### DNS записи
Добавьте следующие записи в вашем DNS провайдере:

```
Тип  Имя   Значение              TTL
A    @     ваш-IP-адрес          3600
A    mail  ваш-IP-адрес          3600
MX   @     mail.alexol.io        3600  (приоритет 10)
TXT  @     v=spf1 mx ~all        3600
```

### SSL сертификаты
```bash
# Установите certbot
sudo apt-get install certbot

# Получите сертификат
sudo certbot certonly --standalone -d alexol.io -d mail.alexol.io
```

### Брандмауэр
```bash
# Откройте необходимые порты
sudo ufw allow 25/tcp    # SMTP
sudo ufw allow 587/tcp   # SMTP Submission
sudo ufw allow 143/tcp   # IMAP
sudo ufw allow 80/tcp    # HTTP
sudo ufw allow 443/tcp   # HTTPS
```

## Устранение неполадок

### Не могу войти в админ-панель
```bash
# Проверьте логи backend
docker-compose logs backend

# Пересоздайте базу данных
docker-compose down -v
docker-compose up -d
```

### Письма не отправляются
```bash
# Проверьте SMTP сервер
docker-compose logs backend | grep SMTP

# Проверьте порты
netstat -tulpn | grep -E '25|587'
```

### Outlook не подключается
1. Проверьте правильность настроек
2. Убедитесь, что указали полный email в логине
3. Проверьте, что сервер доступен: `telnet localhost 143`
4. Проверьте логи: `docker-compose logs backend`

### Не загружается аватар
```bash
# Проверьте MinIO
docker-compose logs minio

# Откройте консоль MinIO
http://localhost:9001
```
