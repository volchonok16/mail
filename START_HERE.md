curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

## 🆘 Если что-то не работает

### Не запускается?
1. Убедитесь, что Docker Desktop запущен
2. Проверьте, что порты свободны (3000, 8000, 25, 587, 143)
3. Перезапустите Docker Desktop
4. Попробуйте снова

### Не могу войти?
- Используйте дефолтный логин: `admin@alexol.io` / `Gord078134Alexol!9256`
- Если изменили и забыли - пересоздайте БД: `docker-compose down -v && docker-compose up -d`

### Письма не отправляются?
- Проверьте логи: `docker-compose logs backend`
- Проверьте веб-интерфейс (должен работать)
- См. TESTING.md для детальной диагностики
