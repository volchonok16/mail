## 🎯 Как это работает:

### Для администратора:
```
1. Админ входит (admin@alexol.io)
   ↓
2. Показывается попап:
   ┌────────────────────────────────┐
   │  Куда вы хотите перейти? [X]  │
   │  ┌──────────┐  ┌──────────┐   │
   │  │    👥    │  │    📧    │   │
   │  │Управление│  │Моя почта │   │
   │  └──────────┘  └──────────┘   │
   └────────────────────────────────┘
   ↓                    ↓
/admin              /dashboard
```

### Для обычного пользователя:
```
1. Пользователь входит
   ↓
2. Сразу переходит в /dashboard
   (попап НЕ показывается)
```

### Переключение для админа:
```
В /admin:
  [📧 Моя почта] - переход в почту

В /dashboard (если админ):
  [👥 Управление пользователями] - переход в админку
```
