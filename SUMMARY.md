## 🎯 Быстрые действия

### Чтобы применить обновления:

```bash
# Вариант 1 (самый простой):
Дважды кликните: rebuild-force.bat

# Вариант 2 (вручную):
docker-compose down
docker-compose up -d --build
```

### Чтобы проверить работу:

1. Откройте http://localhost:3000/admin
2. Вы увидите новые кнопки напротив пользователей:
   - ✏️ Редактировать
   - 🛡️ Сделать админом
   - 🔒 Снять права админа
